# 🐷 Piggy Hunt Adventure

A beautiful 3D gyroscope-powered treasure hunting game built with HTML5, CSS3, Vanilla JavaScript, Three.js, WebGL, and Progressive Web App technologies.

Explore magical worlds, discover hidden piggies, collect rewards, level up, unlock new environments, and become the ultimate Piggy Hunter.

---

## Features

### Core Gameplay

* Explore immersive 3D environments
* Hunt hidden piggies
* Capture piggies for rewards
* XP and Level progression
* Coin and Gem economy
* Collection Book system
* Daily Rewards

### Controls

#### Desktop

| Action        | Control        |
| ------------- | -------------- |
| Move          | WASD           |
| Look Around   | Mouse          |
| Sprint        | Shift          |
| Pause         | ESC            |
| Fullscreen    | Double Click   |
| Capture Piggy | Capture Button |

#### Mobile

| Action        | Control        |
| ------------- | -------------- |
| Look Around   | Gyroscope      |
| Move          | Touch Controls |
| Capture Piggy | Capture Button |

## GitHub Pages Deployment

1. Create Repository:

Piggy-Hunt-Adventure

2. Push Source:

git init
git add .
git commit -m "Initial release"
git branch -M main
git remote add origin YOUR_REPO_URL
git push -u origin main

3. Enable GitHub Pages:

Repository Settings
→ Pages
→ Deploy from Branch
→ main
→ root

Your game will be available at:

https://YOUR_USERNAME.github.io/Piggy-Hunt-Adventure/
